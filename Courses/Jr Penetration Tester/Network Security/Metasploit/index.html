    TryHackMe - Metasploit Walkthrough

    Task 1: Nmap Scan

    For this task, we started by running an Nmap scan on the target machine using the following command:

    nmap -sC -sV -p- -T4 --min-rate=9326 -vv [MACHINE IP]

    The options used include:

    -sC: Run particular scripts on the target.
-sV: Check for versions of services.
-p-: Scan all ports.
-T4: Set the scan speed to level 4.
--min-rate=9326: Specify the packet sending rate.
-vv: Enable very verbose output.

    Results:

    Question 1: How many ports are open on the target system?
    Answer: 5

Question 2: Using the relevant scanner, what NetBIOS name can you see?
    Answer: ACME IT SUPPORT

Question 3: What is running on port 8000?
    Answer: WEBFS/1.21

    Task 5: Exploitation

    nmap -sC -sV -p- -T4 — min-rate=9326 -vv [MACHINE IP]

    Refer to previous information about the Windows 7 Professional service. Use the following exploit:

    Exploit: windows/smb/ms17_010_eternalblue
Payload: generic/shell_reverse_tcp

    Question 1: Exploit one of the critical vulnerabilities on the target VM.
